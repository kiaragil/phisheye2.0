S.U.N.N.

WEB DEV
S.U.N.N. is a San Francisco organization dedicated to empowering residents through garden education and food literacy. I was approached to develop and launch a website that was informative and accesible to all. I also helped developed their logo that appears throughout the site.

UniTea

SOFTWARE DEV/WEB DEV
UniTea is a web app developed for a final group project. It's purpose was to promote university study groups in a post-COVID setting. Users could make accounts, make and interact with posts, and even customize their profile. I acted as a member of the back-end team and as project lead. It was hosted using AWS EC2.

Hoply

SOFTWARE DEV/WEB DEV
Hoply is a project made to practice CRUD operations. It is is a very simple text-based social media. Users were able to create an account and then make, edit, react to- posts. I developed the app on Android studio with Java and SQL.
